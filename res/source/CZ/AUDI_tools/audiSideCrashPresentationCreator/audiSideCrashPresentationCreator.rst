audiSideCrashPresentationCreator
================================

Creates standard post-processing presentation for the side crash based
on parameters given in the input control_file.ctrl. To create a template
of this control file use -template parameter.

Usage
-----

usage::

    audiSideCrashPresentationCreator [-h] [-template] [-doc] [controlFileName]
    
    positional arguments:
      controlFileName  Control file name.
    
    optional arguments:
      -h, --help       show this help message and exit
      -template        Creates a control file example.
      -doc             Shows documentation.
