fileTools
=========

Provides multiple types of file modification performed on a list of selected files in the single step.

Available file modification
---------------------------

* Clean ABAQUS files
* Compress files
* Convert to metadb
* Delete files
* Export includes
* Rename files
* Replace file content

Usage
-----

usage::

    fileTools [input parameter]

example::

    fileTools "input parameter"
