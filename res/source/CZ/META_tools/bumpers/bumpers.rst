Bumpers postprocessing toolbar
==============================

The toolbar contain various scripts/session which automatize/support postprocessing of bumper simulations.

There are tools for postprocessing of 
     - PV1200
     - material test
     - EHK42
     - local stiffness

Location
--------

META > User Toolbars > Bumper_toolbar


Requirements
------------

.. warning::
    in the folder is list of actual configuration sessions /data/fem/+software/SKRIPTY/tools/META_session/bumpers/default/bin/SetUp_files
    
* designed for META V19.1.1
