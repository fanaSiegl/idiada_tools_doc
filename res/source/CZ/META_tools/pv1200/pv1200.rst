PV1200 evaluation toolbar
=========================

The script make evaluation of change of gaps usualy after the temperature test

* base on user defined configuration file and user sets 
* define the change distace in local coordinate system between two states


Location
--------

META > User Toolbars > PV1200


Requirements
------------

.. warning::
    
    The toolbar refer to META_gaps_evaluation script only
    
* designed for META 18.1.1 or later
