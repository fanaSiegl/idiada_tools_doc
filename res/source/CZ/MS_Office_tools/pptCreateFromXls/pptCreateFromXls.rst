pptCreateFromXls
================

Python script for generation of Powerpoint presentation based on CSV or Excel, or updating existing presentation with new pictures 

Usage
-----

| PPT CREATION:
| pptCreateFromXls -example
| pptCreateFromXls [-d DELIMITER] -c CSV_SOURCE [PRESENTATION_NAME]
| pptCreateFromXls -x EXCEL_SOURCE [PRESENTATION_NAME]
|
| PPT UPDATE:
| pptCreateFromXls -a POWERPOINT_SOURCE
| pptCreateFromXls -p POWERPOINT_SOURCE -f FOLDER_WITH_PICTURES [PRESENTATION_NAME]
| pptCreateFromXls -p POWERPOINT_SOURCE -u PPT_ANALYSIS_EXCEL [PRESENTATION_NAME]

Example
--------

| PPT CREATION:
| pptCreateFromXls -example                                         # Download example Excel file
| pptCreateFromXls -x source.xlsx preview.pptx                      # Excel file as input
| pptCreateFromXls -c source.csv preview.pptx                       # CSV with default delimiter (comma)
| pptCreateFromXls -d ';' -c source.csv preview.pptx                # CSV with delimiter ;
| pptCreateFromXls -t                                               # Test run with default CSV
|
| PPT UPDATE:
| pptCreateFromXls -a source.pptx                                   # Analyze shapes of Powerpoint presentation
| pptCreateFromXls -p source.pptx -f images/ preview.pptx           # Powerpoint presentation and folder with shapes as input  
| pptCreateFromXls -p source.pptx -f images/ -u ppt_analysis.xlsx preview.pptx  # Powerpoint presentation and PPT analysis in Excel as input  


Description
-----------

* Based on chosen input performs:
    * -c, --csv:        Presentation is created from data provided in CSV file
    * -x, --xls:        Presentation is created from data provided in Excel file
    * -a, --analyse:    Analyses shapes in provided presentation
    * -p, --ppt:        Pictures in provided presentation are updated by ones in given folder
    * -u, --update:     Updates Alt_Text fields in given presentation from provided Excel file
* Given output file name should have suffix .pptx. If not provided, the suffix is added automatically
* Default output file name is YYYYMMDD_Presentation.pptx, where YYYYMMDD represents today's date
* Currently only pictures are supported for insertion
* Delimiter option (-d, --delimiter) is taken into acocunt only if option --csv is used
* If no folder provided for option --ppt, current working directory is checked for the images
* For update option the updated Excel file generated by --analyse option must be provided

Warning
-------
In case the picture was copy-pasted in Powerpoint and saved, they refer to the same image and update by the script results in change of all instances of the picture (being worked on)
