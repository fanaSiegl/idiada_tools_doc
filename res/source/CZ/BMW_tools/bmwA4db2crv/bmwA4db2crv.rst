BMW a4db 2 crv
===============
Converts new format *.a4db.crv binary file to old format *.crv text file.

Usage
-----
bmwA4db2crv DIR/FILE.a4db.crv

Description
-----------
* based on script from BMW (/share/eg-3-cae/Postprocessing/a4db2crv/)
* requires new format *.a4db.crv binary file
* outputs old format *.crv file text file
