bmwSimReportAssistant
=====================

Python script for preparing a presentation for BMW StdAuswertung SimReport 

Usage
-----

bmwSimReportAssistant <INPUTDECK_NEW_VARIANT>

bmwSimReportAssistant <INPUTDECK_NEW_VARIANT> <INPUTDECK_BASIS>

Example
--------

bmwSimReportAssistant G70_E_VL-VKBG_CH301D80R_H05EG001_84kmh

bmwSimReportAssistant G70_E_VL-VKBG_CH301D80R_H05EG001 G70_E_VL-VKBG_CH301D80R_H05EG011 

Description
-----------

* prepares configuration of the PPTX file for SimReport post-processing
* names a presentation with current date and input deck names
* prints current working directory in Windows format for easy navigation
