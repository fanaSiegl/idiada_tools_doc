newBMWvariant
=============

Creates a new BMW variant project structure and copies data from sync if present.

Usage
-----

newBMWvariant [-h] variantName [ticket number]

positional arguments:
  variantName   - Complete name of a variant.
  
  ticket number - Ticket number.

optional arguments:
  -h, --help     show this help message and exit

Description
-----------

* creates a new BMW variant project structure according to variant name
* copies files if present in sync directory
