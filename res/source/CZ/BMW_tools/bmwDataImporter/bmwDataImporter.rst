bmwDataImporter
===============

Python script for based on cb2tools for BMW data import.

Usage
-----

bmwDataImporter

Description
-----------

* according to a given variant name searches for available includes, result files and variant info if any
* exports selected files to BMW file system
* creates a new local directory structure base on variant name
* imports data from BMW to local file system
