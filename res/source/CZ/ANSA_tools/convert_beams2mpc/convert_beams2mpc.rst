Converts beam to MPC
====================

Convert the BEAM cluster to MPC

* only for visible BEAMs
* works for ABAQUS
* convert to MPC

Location
--------

Tools>convert_beams2mpc

Usage
-----

* select BEAM custers which you want convert to MPC

Requirements
------------

.. warning::
    
    Tested only for ABAQUS
    
* designed for ANSA V.19.1.1 or later version
