bmwCableScript
==============

Converts CAD cable geometry to BEAMS.

Location
--------

BMW>beamCableCreation

Usage
-----

* select cable property to be converted into BEAMS
* select first CON/curve going along the cable faces (use Feature Line selectio option)
* select second CON/curve going along the cable faces (use Feature Line selectio option)

Requirements
------------

.. warning::
    
    No special requirements.
    
* designed for ANSA V.19.1.1
