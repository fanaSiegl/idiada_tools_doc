coorSys4gaps
=========

is a tool to create local coordinate systems for gap measurement in META

Usage
-----

1. select guiding edge using Feature Line in Feature Selection
2. select opposite edge the same way
3. add or modify guiding coordinate systems that are automatically created
4. select file name for META configuration file, line name and line color

.. warning::

    Please be aware that node sequence selection matters! Tthe first selected node must be the first on the edge.
