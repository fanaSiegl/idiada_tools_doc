BMW Conn Include
=================

ANSA button for automatic creation of conn include according to BMW MOKA requirements.

* GUI ANSA button 
* user selectes slave nodes
* user selectes master edges
* user selects pid, that will be used for truss elements
* all entities required for conn include (truss property, sets, RB, ties, cont excl, step) are created either in new include, or in slave (nodes) include

icons used: `https://www.flaticon.com/packs/essential-collection <https://www.flaticon.com/packs/essential-collection/>`_.
