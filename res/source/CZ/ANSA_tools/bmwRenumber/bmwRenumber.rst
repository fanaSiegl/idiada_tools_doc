BMW Renumber
==============

ANSA button for renumbering according to BMW MOKA requirements.

* GUI ANSA button 
* User selects numbering range from MOKA numbering convention excel datasheet.
* Python renumbers the entities in ANSA.
* Python creates cb2header as comment
* If T-ALL is missing, python will create it.
