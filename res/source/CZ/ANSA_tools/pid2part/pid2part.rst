pid2part
========

Creates ANSA PART structure based on PID name for easier orientation in the model. New created parts are organised into subassemblies. Just switch ANSA Draw mode to PART.

Usage
-----

just click the button.

Description
-----------

* searches PID names
* creates new PART entites
* assigns PIDs to these parts
