Loss factor estimation
======================

Half power bandwidth method

Usage
-----

loss_factor_estimation

Description
-----------
* open dialog added 
* requires *.txt file with frequency response curve
* estimates the loss factor coefficients by using the half-power bandwidth method
* provides the loss factor coefficient estimation for given frequencies (the results should be used only for information, half-power andwidth methon is not suitable for lot of cases)
